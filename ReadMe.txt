# Heuristic-algorithm



This Toolset comprised of the following files

1)Unscheduled_states.m
Initial and unscheduled states of the given appliances


2)Appliances_data.m
The appliance data including types,power rating, time span, maximum and minimun power etc.
 

3)init_gen_1.m
Generates the initial population for GA.

4)Example_of_GA.m
Performs the steps of GA.

5)crossover.m and mutate.m
These files are used to perform stochastic operatoration steps i.e., crossover and mutation

6)f_val_final_v1.m
Evaluate fitness function
